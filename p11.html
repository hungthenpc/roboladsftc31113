ARDUINO TUTORIAL #11: SERIAL COMMUNICATION

19/01/2026 Lập Trình Độ khó: Cơ bản
Arduino Serial Communication Debug

MỘT SỐ LỆNH LIÊN QUAN ĐẾN SERIAL

1. Serial.begin(speed);

Tác dụng: Mở cổng Serial port, set bits per seconds to speed (thường là 9600).

Vị trí: Được đặt ở trong void setup() {}.

2. Serial.print(content);

Tác dụng: In ra liên tiếp các dòng (lưu ý không có xuống dòng) mà giá trị content được điền vào.

Lưu ý: Có thể dùng Serial.println(content); để in ra có xuống dòng.

BÀI TẬP: SỬ DỤNG KIẾN THỨC Ở TRÊN VÀ THỰC HÀNH GÕ NHỮNG DÒNG LỆNH SAU

Code bài tập: Sử dụng Serial để in ra năm và tuổi.

Ví dụ code:

String mess1 = "In ";
int year = 2008;

String mess2 = " I would be ";
int age = 0;

String mess3 = " years old";
int wait = 750;

void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.print(mess1);
  Serial.print(year);
  Serial.print(mess2);
  Serial.print(age);
  Serial.println(mess3);
  year=year+1;
  age=age+1;
  delay(wait);
}

KẾT QUẢ MONG ĐỢI:

Kết quả hiển thị trên Serial Monitor.
